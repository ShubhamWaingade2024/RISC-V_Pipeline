# RISC-V Pipeline

RISC-V Pipelined Processor is a 32-bit processor designed in Verilog HDL using a 5-stage pipeline (IF, ID, EX, MEM, WB) to improve instruction throughput. It implements pipeline registers, hazard detection, and data forwarding for efficient execution. The design was simulated and verified using Icarus Verilog, GTKWave, and VS Code.

## Structure
- `code/` contains the pipeline modules and testbench
- `images/` contains project screenshots or diagrams

## Getting started
1. Open the Verilog sources in the `code/` directory.
2. Run the testbench with your preferred simulator.
3. Review the waveform output for pipeline behavior.

## Notes
This project is intended as a learning-focused implementation of a RISC-V pipeline in Verilog.
